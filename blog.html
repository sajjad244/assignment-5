1. What is the Document Object Model (DOM)?

The DOM represents documents as a tree for dynamic manipulation.

2. How do you select an element from the DOM?

You can select an element using methods like `getElementById()`, `querySelector()`, or `getElementsByClassName()`.

3. What is event delegation in the context of the DOM, and why is it useful?

Handle events at a parent level; improves performance and efficiency.

4. How do you manipulate an element's attributes and styles using the DOM?

Use element.setAttribute() and element.style.property for manipulation.
